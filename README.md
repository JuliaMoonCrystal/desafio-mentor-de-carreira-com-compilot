## 🔝 Top 3 Carreiras em Tecnologia para Você

Com base nas suas respostas, aqui está o ranking das 3 carreiras mais alinhadas ao seu perfil:

* **Sinergia:** Une o interesse por dados e código.

### 3. Desenvolvedora Backend
* **Sinergia:** Foco em código e construção de sistemas robustos.
* **Foco:** Boa opção para quem gosta de criar produtos e quer evoluir tecnicamente.

## 🧩 VISÃO DO DIA A DIA
**Como é o trabalho de uma Engenheira de Dados:**

- Construir pipelines de dados para coleta e transformação
- Modelar bancos de dados relacionais e NoSQL
- Garantir qualidade e integridade dos dados
- Otimizar performance de consultas e sistemas
- Colaborar com cientistas de dados e analistas de negócio

## 🚀 PROJETO DE PORTFÓLIO
### PROJETO: Data Pipeline de Vendas

**O QUE FAZER:** Criar um pipeline que coleta dados de vendas (CSV ou API), transforma em formato limpo e carrega em um banco de dados.

#### Entregáveis:
- Script de ingestão em Python
- Banco de dados populado (PostgreSQL)
- Dashboard simples com métricas

#### Critérios de Aceitação:
- Pipeline automatizado
- Dados consistentes e limpos
- Documentação clara

> 💡 **DICA:** Use dados públicos de e-commerce ou vendas fictícias para simular.

## 💬 ROTEIRO DE ENTREVISTAS

1. **PERGUNTA 1:** *"Explique um projeto de dados que você desenvolveu."* **COMO RESPONDER:** Descreva objetivo, ferramentas usadas e impacto.
2. **PERGUNTA 2:** *"Qual a diferença entre banco relacional e NoSQL?"* **COMO RESPONDER:** Relacional = tabelas estruturadas; NoSQL = flexível para dados não estruturados.
3. **PERGUNTA 3:** *"Como garantir qualidade dos dados em um pipeline?"* **COMO RESPONDER:** Validações, testes automatizados e monitoramento.
4. **PERGUNTA 4:** *"Explique o conceito de ETL."* **COMO RESPONDER:** Extrair dados, transformar em formato adequado e carregar em destino.
5. **PERGUNTA 5:** *"Como você otimiza consultas SQL?"* **COMO RESPONDER:** Índices, normalização e análise de planos de execução.
